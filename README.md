# 🏥 Hospital Emergency Room Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive Hospital Emergency Room Dashboard developed in Power BI to analyze emergency department operations. The dashboard provides insights into patient visits, waiting times, admission trends, demographics, and departmental performance, helping healthcare administrators improve operational efficiency and patient care.

---

## 🎯 Project Objectives
- Monitor emergency room patient visits.
- Analyze patient waiting times.
- Track admission and discharge rates.
- Identify peak patient traffic by day and time.
- Understand patient demographics.
- Support data-driven decision-making for hospital management.

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel (Dataset)

---

## 📊 Dashboard Features
- Total Emergency Room Visits
- Average Waiting Time
- Admission vs. Discharge Analysis
- Patient Satisfaction (if available)
- Visits by Age Group
- Gender Distribution
- Department-wise Patient Count
- Referral Source Analysis
- Peak Hour Analysis
- Monthly ER Visit Trends
- Interactive Filters and Slicers

---

## 📂 Dataset
The dataset includes anonymized emergency room patient records for analytical purposes.

Sample columns:
- Patient ID
- Visit Date
- Age
- Gender
- Department
- Waiting Time
- Admission Status
- Referral Source
- Diagnosis Category

---

## 📈 Key Insights
- Highest patient volume occurs during evening hours.
- Adults aged 18–45 account for the majority of ER visits.
- Waiting times increase during weekends.
- Most patients are discharged after treatment.
- Emergency Medicine handles the highest number of cases.
